== Installation ==

1. Upload the plugin files to the /wp-content/plugins/ directory, or upload the .zip file with the plugin under **Plugins > Add New > Upload.**
2. Activate the plugin through the ‘Plugins’ screen in WordPress
3. You can find Premium Addons Elements under the category **MAMA WIDGETS on your Elementor element/widget list.
